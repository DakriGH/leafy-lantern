# Leafy-Lantern — gioca nel browser

Questa è la **versione compilata** del gioco, pubblicata per poterci giocare da
qualsiasi dispositivo (Chromebook compresi) senza installare nulla.

▶️ **https://dakrigh.github.io/leafy-lantern/**

Si può anche "installare" come app: nel browser, menu → *Installa app*. Si apre
a schermo intero con la sua icona.

## Note
- Il **sorgente** vive in un repository separato: qui c'è solo il risultato
  della build (`scripts/prepara-www.mjs`).
- Serve HTTPS perché la **camera** (realtà aumentata) funzioni: questo indirizzo
  ce l'ha.
- Il gioco è pensato prima di tutto per il **tocco**: joystick a sinistra, il
  tasto salto e la bolla degli strumenti a destra. Sulla bolla: **tocca** per
  usare quello che hai in mano, **trascina** per aprire la ruota.
